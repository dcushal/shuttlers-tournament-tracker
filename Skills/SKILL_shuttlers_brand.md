# Shuttlers Tournament Tracker - Brand Guidelines

## Overview
Badminton club management app for "8:30 Shuttlers" - tracking daily games 20:30-22:30. Dark mode UI with green accent (#9ee312), premium sports aesthetic.

## Brand Attributes
- **Personality:** Bold, competitive, community-driven
- **Vibe:** Premium sports app, not casual
- **Audience:** Serious badminton players

## Color Palette
| Role | Color | Hex |
|------|-------|-----|
| Primary | Electric Green | #9ee312 |
| Background | Deep Black | #000000 |
| Surface | Zinc 900 | #18181b |
| Surface Light | Zinc 800 | #27272a |
| Text Primary | White | #ffffff |
| Text Secondary | Zinc 400 | #a1a1aa |
| Error | Red 500 | #ef4444 |
| Success | Green 500 | #22c55e |

## Typography
- **Headlines:** Black weight, uppercase, tight tracking
- **Body:** Regular weight, readable
- **Accents:** Bold for stats/numbers

## Design Patterns
- Cards with subtle borders (green at 20% opacity for premium feel)
- Rounded corners (2xl-3xl)
- Dark theme throughout
- Green accent for CTAs and highlights

## Icon Style
- Lucide React icons
- Stroke-based, 2px weight
- Green for active states

## Quick Reference
- Green = Primary action/success
- Dark backgrounds = Main surfaces
- Uppercase = Labels/headers
- Numbers = Bold black
